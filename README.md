# Code Academy - "Become the Programmer Who Saves the School"

**Code Academy** adalah game edukasi berbasis web interaktif bergenre *Adventure, Puzzle,* dan *RPG Lite*. Proyek ini dirancang khusus untuk siswa kelas X PPLG (Pengembangan Perangkat Lunak dan Gim) guna mempelajari konsep dasar pemrograman dengan cara yang seru, menantang, dan jauh dari kesan kuis konvensional.

---

## 🎮 Gambaran Umum Game

Saat orientasi dimulai, seluruh sistem sekolah diserang oleh **Bug King**. Setiap ruangan di sekolah mengalami kerusakan. Tugas Anda sebagai siswa baru adalah menjelajahi sekolah menggunakan tombol WASD atau Arrow Keys, menyelesaikan berbagai tantangan logika, mengalahkan Bug King, dan menyelamatkan sekolah!

* **Genre:** Adventure • Puzzle • Educational • RPG Lite
* **Platform:** Web Browser (HTML, CSS, JavaScript)
* **Target Pemain:** Siswa kelas X PPLG
* **Estimasi Durasi Main:** 5–10 menit

---

## ✨ Fitur Utama

* **Custom Character Creator:** Pemain dapat mengubah penampilan karakter mulai dari rambut, hoodie, warna pakaian, hingga nama karakter.
* **8 Level Tantangan Logika:**
    1. *Welcome Programmer* – Pengenalan sekolah dan interaksi dengan NPC.
    2. *Variable Room* – Drag & drop data ke kotak variabel.
    3. *Input Output* – Pencocokan logika input dan output.
    4. *If Else* – Pengambilan keputusan yang benar untuk membuka akses pintu.
    5. *Loop Factory* – Mengontrol robot menggunakan perintah perulangan (*Repeat 5x*).
    6. *Debug Room* – Menemukan *bug* pada potongan kode bergambar.
    7. *Function Machine* – Menyusun urutan alur fungsi (*Start → Input → Process → Output → End*).
    8. *Boss Battle* – Pertarungan akhir melawan Bug King di mana jawaban yang benar akan mengurangi HP boss.
* **Sistem Score & Combo:** Perhitungan skor dinamis dengan bonus Combo/Perfect, serta penalti pengurangan skor jika menggunakan *Hint*.
* **Achievement & Badge System:** Penghargaan menarik seperti *Bug Hunter* hingga *Coding Master* berdasarkan total skor yang diperoleh.
* **Sertifikat Digital:** Di akhir permainan (Ending), sistem akan menampilkan total skor, akurasi, waktu, serta memberikan sertifikat digital yang bisa diunduh oleh pemain.

---

## 🚀 Teknologi yang Digunakan

Proyek ini dibangun menggunakan teknologi *native web* untuk memastikan performa yang ringan, responsif, dan mudah dijalankan:

* **HTML5 & CSS3** – Untuk struktur halaman, tema warna modern (Biru, Ungu, Cyan), animasi *glow*, *particle effect*, hingga efek *screen shake* saat melawan boss.
* **Vanilla JavaScript** – Menangani seluruh logika permainan, state management, dan alur gameplay dari awal hingga selesai.
* **Web API (LocalStorage)** – Digunakan untuk menyimpan data progres, skor, dan karakter pemain secara lokal.
* **requestAnimationFrame()** – Digunakan untuk menangani rendering animasi dan pergerakan karakter secara mulus.
